misaki.html.conv/date->string で好みのフォーマットを指定

config.clj に好みの日付フォーマットを指定できるようにしたい。

一旦、下記のようなコードで対処。_config.clj には、:date-format "yyyy-MM-dd" のように記述しておく。デフォルト:date-formatのデフォルト値は:simple。 

; =date->string
(defn date->string
  "Convert org.joda.time.DateTime to String

      (date->string (clj-time.core/now))
      ;=> \"01 Jan 2012\""
  [date]
  (if date
    (if (= (:date-format *site*) :simple)
      (str/join
       " "
       [(.toString date "dd")
        ; Joda: "MMM" is not worked in my environment
        (nth '(_ Jan Feb Mar Apr May Jun Jul Aug Sep Oct Nov Dec)
             (t/month date))
        (.toString date "yyyy")])
      (.toString date (:date-format *site*)))))

_config.clj には :site に設定した。がこれは美しくない(と思 う)。_config.clj の toplevel(?) に本当は設定すべきだと思うのだが、うま くいかなかった...。 

:site {
        ;; :
        ;; :

        }
